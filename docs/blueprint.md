# **App Name**: Suna's Portfolio

## Core Features:

- Biography Section: Showcase a brief biography in Turkish, highlighting key accomplishments and experiences.
- Gallery Section: Display a gallery of photos or videos reflecting personal interests and experiences.
- Links Section: Link to her blog and LinkedIn page
- Personalized Turkish Proverbs: Include a tool for generating personalized Turkish proverbs based on user input.
- Contact Section: A contact section, offering visitors to write email messages to the user. Handle contact form submissions directly via email without storing them in a database.
- Hosting: Github pages compatibility.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a gentle, calming feel.
- Background color: Light gray (#F5F5F5) to ensure readability and a clean aesthetic.
- Accent color: Muted gold (#B8860B) to highlight important information and calls to action.
- Font pairing: 'Alegreya' (serif) for headlines and 'PT Sans' (sans-serif) for body text. 'Alegreya' will bring an elegant and intellectual feel to headlines, while 'PT Sans' ensures readability and a modern touch for body text.
- Use simple, elegant icons to represent different sections and interests. These icons should be easily recognizable and aesthetically pleasing.
- Design a clean, intuitive layout with clear sections and a focus on easy navigation. Ensure the site is responsive and looks great on all devices.
- Incorporate subtle animations and transitions to enhance user engagement and create a smooth browsing experience. Animations should be used sparingly to avoid distracting from the content.
